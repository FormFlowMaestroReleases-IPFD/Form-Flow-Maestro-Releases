# Form Flow Maestro Releases

This repository contains release artifacts for Form Flow Maestro Chrome Extension.

> **Note**: This is a private repository. You must have GitHub access and be logged in to download releases.

## Structure

- `releases/` - Versioned ZIP files ready for distribution
- `content/` - Content for this readme

## Installation

1. Go to `releases/` directory
2. Download the latest version ZIP file
3. Unzip the file
4. Open Chrome ‚Üí `chrome://extensions/`
5. Enable "Developer mode"
6. Click "Load unpacked"
7. Select the unzipped folder

## Updating

To update to a new version:

1. Download the new version ZIP from the [`releases/`](releases/) directory
2. Unzip the new version
3. Open Chrome ‚Üí `chrome://extensions/`
4. Find "Form Flow Maestro" extension
5. Click the refresh/reload button on the extension card
6. Or remove the old version and follow installation steps with the new ZIP
